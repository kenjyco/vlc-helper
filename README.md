You must have the following system packages installed

- `vlc`
- `imagemagick`
- `wmctrl`

You must also have the `dbus-python` package in your system-wide Python
installallation.

## Install

```
% python3 -m venv venv
% venv/bin/pip3 install vlc-helper
% python3 -m venv --system-site-packages venv
```
